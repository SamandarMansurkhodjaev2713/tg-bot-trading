# Пайплайн исправления `complete_fix.py`

## Архитектура

- `complete_fix.py` — CLI-скрипт, безопасно заменяющий функцию `aitrader_command` через AST.
- `fixer/` — модульный пакет:
  - `exceptions.py` — исключения пайплайна
  - `file_ops.py` — надёжные операции чтения/записи
  - `patcher.py` — AST-патчер функции
- `tests/` — модульные тесты `unittest` для критических операций

## Запуск

```bash
python complete_fix.py
```

По умолчанию обрабатывает `simple_forex_bot.py` в текущей директории.

## Тестирование

```bash
python -m unittest -q
```

## Требования

- Python 3.10+
- Стандартная библиотека (без внешних зависимостей)

## Принципы

- SOLID/DRY: разделение ответственности, отсутствие дублирования
- Безопасность: бэкап перед записью, атомарная замена
- Читаемость: PEP8, docstrings, комментарии в сложных местах
